# Connect Four

Simple Connect Four game made in Java.

The project was created mainly to practice:
- object-oriented programming
- game logic
- working with arrays and conditions in Java

---

## Features

- 2-player gameplay
- win detection
- horizontal, vertical, and diagonal checks
- simple and readable code structure

---

## Technologies

- Java

---

## How to Run

Clone the repository:

```bash
git clone https://github.com/MKonrad-afk/GAMEConnectFour.git
```

Open the project in IntelliJ IDEA (or any Java IDE) and run:

```bash
Main.java
```
